## Features :package:

-   Supported Desktop environments:
    -   gnome
    -   kde
    -   xfce
    -   cinnamon
-   Bluetooth support
-   Printing support
-   Flatpak support

## Usage :rocket:

1. Run the following command:

    ```bash
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/damienbutt/arch-de/HEAD/scripts/arch-de.sh)"
    ```

2. Follow prompts until the setup is complete.
